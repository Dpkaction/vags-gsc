GSC Coin Wallet - Standalone Executable
======================================

This package contains:
- GSC_Coin_Wallet.exe - Main wallet application
- Install_GSC_Wallet.bat - Optional installer

Features:
- Complete GSC blockchain functionality
- 21.75 trillion total supply
- Bitcoin-like reward halving system
- Professional wallet interface
- Mining capabilities
- Transaction processing
- QR code generation
- Address management

To run:
1. Double-click GSC_Coin_Wallet.exe
   OR
2. Run Install_GSC_Wallet.bat for system installation

No additional software required - all dependencies included!

System Requirements:
- Windows 7 or later
- 100 MB free disk space
- Internet connection (optional, for future updates)

Support:
- This is a standalone cryptocurrency wallet
- Keep your private keys and seed phrases safe
- Always backup your wallet data
